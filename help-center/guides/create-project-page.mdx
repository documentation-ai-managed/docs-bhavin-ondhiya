---
title: Creating a New Project Page
description: Quick guide to add a new page for your project documentation.
---

## Steps

<Steps>
  <Step title="Open your project">
    ````
    Go to your dashboard and select your desired project.
    ````
  </Step>
  <Step title="Click 'Add Page'">
    ````
    Find the 'Add Page' button and click to start a new page.
    ````
  </Step>
  <Step title="Name and write content">
    ````
    Enter a title and use the editor to add your documentation.
    Save as draft or publish when ready.
    ````
  </Step>
</Steps>
