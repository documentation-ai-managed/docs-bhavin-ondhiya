## Overview

Bhavin Ondhiya provides a centralized space for your project documentation. You can organize pages, collaborate with teams, and publish polished docs with ease. This platform handles everything from simple READMEs to comprehensive API references.

Use the sidebar navigation to explore topics. Start with the [quickstart guide](/quickstart) to set up your first space.

<Callout kind="info">
  Bhavin Ondhiya uses brand color `#3B82F6` for primary elements. Apply it consistently in your custom themes.
</Callout>

## Brand Guidelines

Follow these guidelines to maintain consistency across your documentation.

- **Primary Color:** `#3B82F6` – Use for buttons, links, and accents.
- **Typography:** Sans-serif fonts with clear hierarchy (H2 for sections, H3 for subsections).
- **Icons:** Lucide library – Select from `book-open`, `code`, `zap`, and others.

<Expandable title="Advanced Theming" default-open="false">

Customize further with CSS variables:

````css
:root {
  --brand-primary: #3B82F6;
  --brand-secondary: #1E40AF;
}
````

Apply in your space settings.

</Expandable>

## Quick Start Guide

Get your documentation live in minutes.

<Steps>
  <Step title="Create Account" icon="user-plus">

  Visit [https://bhavinondhiya.com](https://bhavinondhiya.com) and sign up with your email.

  </Step>
  <Step title="New Space" icon="plus">

  Click "New Space" in the dashboard. Name it "My Project Docs".

  </Step>
  <Step title="Add Page" icon="file-plus">

  Create your first page. Use Markdown for content:

  ````markdown
  # My Project
  Welcome to the docs.
  ````

  </Step>
  <Step title="Publish" icon="globe">

  Hit "Publish". Share the live URL: `https://docs.bhavinondhiya.com/my-project`.

  </Step>
</Steps>
